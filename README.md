# AreWeFastYet

This is a port of Stefan Marr's Are We Fast Yet? 's benchmarking suite.

These benchmarks were originally written in SOM, a minimal Smalltalk dialect, and loaded into Pharo using the original tools.

Usage

    AreWeFastYet run.
